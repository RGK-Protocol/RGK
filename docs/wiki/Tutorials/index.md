## How to Choose a Tutorial

| You want to… | Start at |
| --- | --- |
| See something working in 10 minutes. | Tutorial-0 |
| Understand the philosophy / lineage-first identity. | Tutorial-1 |
| Write code that issues and transfers an asset. | Tutorial-2 |
| Integrate RGK into a wallet. | Tutorial-3 |
| Run the e2e harness. | Tutorial-4 |
| Operate the `rgk-walletd` daemon. | Tutorial-5 |
| Read the spec, not run anything. | [Reference](../Reference/Resolver-State-Machine.md) |
| Understand a concept, not the API. | [Concepts](../Concepts/Identity.md) |

Each tutorial ends with "Where to go next" cross-links. The wiki is
designed to be read in any order; pick the surface that matches what
you're trying to do.
